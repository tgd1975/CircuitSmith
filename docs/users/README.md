# CircuitSmith — user documentation

User-facing docs for CircuitSmith: a linear tutorial and a curated
gallery of worked examples.

| Section | What it covers |
|---|---|
| [Tutorial](tutorial/) | Six-step walkthrough from a minimal `.circuit.yml` to a BOM round-trip (~30 min). |
| [Examples](examples/) | Worked `.circuit.yml` projects with rendered SVGs, BOMs, and short write-ups. |

For developer / contributor documentation (architecture, ADRs, task
system, ERC internals), see [`docs/developers/`](../developers/).
For maker-facing wiring guides shipped alongside the schematic
output, see [`docs/builders/`](../builders/).

> closes, expect the tutorial and gallery directories to fill in
> incrementally.
